# 🚀 AstroReact

AstroReact is an interactive space science learning platform designed to make astronomy and space exploration easier to understand through simulations, research resources, mission tracking, and educational content.

## 🌟 Current Features

* 🔐 User Authentication
* 🏠 Home Page
* 📊 Dashboard
* 🧪 Simulator Lab

  * BMI Calculator
  * Gravity Calculator
  * Ohm's Law Calculator
  * Solar System Explorer
* 🚀 Mission Tracker
* 📰 Space News
* 🌍 Space Agencies
* 📚 Research Papers Hub
* 🔍 Universal Search (Version 1)

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
